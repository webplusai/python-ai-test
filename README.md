# LLM


## For Backend
1) create `.env` file
1) create python env
    ```
    python3 -m venv env
    ```
1) activate python env
    ```
    source env/bin/activate
    ```
1) install requirements
    ```
    pip install -r requirements.txt
    ```

1) start backend API
    ```
    fastapi dev main.py
    ```

1) access doc url
    ```
    http://localhost:8000/docs
    ```

## For Frontend
1) install packages
    ```
    npm install
    ```

1) start frontend
    ```
    npm run dev
    ```
1) access frontend
    ```
    http://localhost:5173/
    ```
